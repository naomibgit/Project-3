## Summary

**AnimalApps** is an Android app where I recreated a selected app screen but transformed it into an adorable animal-themed layout. The project required understanding ConstraintLayout, positioning UI elements, styling, and rebuilding the screen from scratch based on the chosen inspiration screenshot.
